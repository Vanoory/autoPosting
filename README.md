# 🤖 Бот для автоматического ведения торгового Telegram-канала

Бот автоматически генерирует и публикует посты в торговый канал с системой модерации.

## 📋 Возможности

- **Утренние посты**: Каждое утро (8-10 часов) бот генерирует приветственный пост
- **Новостные посты**: 1-2 раза в день генерирует новости о крипте, фьючерсах, ФРС
- **Модерация**: Все посты отправляются админу для одобрения/переписывания
- **Добавление фото**: Можно прикрепить фото к любому посту перед публикацией
- **Человечный стиль**: Посты пишутся естественно, без AI-штампов

## 🚀 Быстрый старт

### 1. Получите необходимые токены

**Telegram Bot Token:**
1. Напишите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям, получите токен

**Ваш Telegram ID:**
1. Напишите [@userinfobot](https://t.me/userinfobot)
2. Скопируйте ваш ID

**ID канала:**
- Если канал публичный: `@channel_name`
- Если приватный: добавьте бота в канал админом, отправьте пост, используйте [@getidsbot](https://t.me/getidsbot)

**Groq API Key (бесплатный):**
1. Зайдите на [console.groq.com](https://console.groq.com)
2. Зарегистрируйтесь
3. Создайте API ключ в разделе API Keys

### 2. Настройте config.py

Откройте `config.py` и замените:
```python
BOT_TOKEN = 'YOUR_BOT_TOKEN_HERE'  # Токен от BotFather
CHANNEL_ID = '@your_channel'        # ID или @username канала
ADMIN_ID = 123456789                # Ваш Telegram ID
GROQ_API_KEY = 'gsk_...'           # API ключ от Groq
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустите бота

```bash
python bot.py
```

## 📱 Использование

### Команды бота (для админа):

- `/start` - Запуск бота и справка
- `/morning` - Сгенерировать утренний пост вручную
- `/news` - Сгенерировать новостной пост вручную
- `/status` - Проверить статус бота

### Процесс модерации:

1. Бот отправляет вам пост с кнопками:
   - ✅ **Одобрить** - публикует пост в канал
   - ✏️ **Переписать** - генерирует новый вариант
   - 📷 **Добавить фото** - позволяет прикрепить изображение

2. Если выбрали "Добавить фото":
   - Отправьте фото боту
   - Появятся кнопки для публикации с фото

## 🌐 Деплой на бесплатный хостинг

### Render.com (рекомендуется):

1. Зарегистрируйтесь на [render.com](https://render.com)
2. Создайте новый Web Service
3. Подключите GitHub репозиторий или загрузите файлы

   - `BOT_TOKEN`
   - `CHANNEL_ID`
   - `ADMIN_ID`
   - `GROQ_API_KEY`
5. Build Command: `pip install -r requirements.txt`
6. Start Command: `python bot.py`

1. Зарегистрируйтесь на [railway.app](https://railway.app)

1. Установите Fly CLI
2. `fly launch`
3. `fly secrets set BOT_TOKEN=... CHANNEL_ID=... ADMIN_ID=... GROQ_API_KEY=...`
4. `fly deploy`

## 🔧 Устранение проблем

**Бот не отвечает:**
- Проверьте правильность BOT_TOKEN
- Убедитесь, что бот запущен (`/start`)

**Не публикует в канал:**
- Добавьте бота в канал как администратора
- Проверьте правильность CHANNEL_ID

**Ошибки генерации:**
- Проверьте GROQ_API_KEY
- Убедитесь, что есть интернет
